How to run the Daily Sleep Tracking  Project
1. copy the  dets folder
2.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
3. Open PHPMyAdmin (http://localhost/phpmyadmin)
4. Create a database with name detsdb 
5. Import detsdb.sql file(given inside the zip package in sql file folder)
6.Run the script http://localhost/dets 
(frontend)
7. click on login/register 
8. register into Sleeptracker then further login
